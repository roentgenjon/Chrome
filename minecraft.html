⚡ Server verbinden

Verbinde per RCON WebSocket Bridge. Starte den Bridge-Dienst auf deinem Nebuliton-Server, dann trage die URL ein.

# Node.js RCON Bridge auf dem Server:
npx ws-rcon --rcon-host 127.0.0.1 --rcon-port 25575 \
  --rcon-password DEINPASSWORT --ws-port 4000
